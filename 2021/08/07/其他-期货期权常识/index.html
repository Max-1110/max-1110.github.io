期货期权常识

发表于 2021-08-07 更新于 2021-08-19 分类于 其他
本文字数： 2.9k 阅读时长 ≈ 3 分钟

期权

分为看涨期权和看跌期权。看涨期权是赌涨，看跌期权是赌跌。期权里股票的最小成交单位是100股。

看涨期权call option—单买

当前苹果股票130美元，想在3个月后购入苹果股票，但担心到时候股票会涨价，就以5美元的权利金购买了看涨期权。3个月后，如果股票跌到了100美元，你可以不行权，以市场价买入，这样只亏权利金5美元。如果股票涨到200美元，如果按市场价买就太贵了，但是如果行权，还是会以130美元买入股票，到时候倒手一卖，这样盈利就是200-130-5=65美元。所以我的利润区间是[-5,+∞)。

而卖我看涨期权的人，最多挣5美元，最多无限亏（因为如果股票涨到1000美元了，还是他负责1000买，然后130美元卖我，他亏了870美元。）

看跌期权put option—单买

手头不持股，以5美元的权利金买入130美元/股的看跌期权，3个月内可以以130美元卖出股票。如果3个月后，股票价格涨到200美元，你手头没股票，也就不必行权了，亏5美元。如果3个月后，股票跌到每股100美元，则可以用市场价100美元买入，再行权以130美元卖出去，则怒挣25美元。如此看来，利润区间是[-5,125]，因为股票最低0美元。

而卖我看跌期权的人，最多挣5美元，最多亏125美元（因为他要承接我130美元卖出去的原价0美元股票）。

看跌期权put option—先买股票再买看跌期权

比如当前持有的苹果股票130美元，你担心未来3个月内苹果股票会跌，就以5美元的权利金购买了看跌期权。3个月后，如果股票涨到了200美元，你可以不行权，以市场价卖出去这样利润是200-130-5=65美元。如果股票跌到100美元，如果按市场价卖就损失了30美元，但是如果行权，还是会以130美元卖出股票，这样亏损就是5美元。所以我的利润区间是[-5,+∞)。

期权的底层资产

指期权的作用物，可以是股票，也可以是期货。

期权可以单独交易

可以5美元的权利金买到期日3个月，行权价130美元的看涨期权。之后这个权利金可能会涨到6美元，那可以直接把这个期权卖出去，赚1美元。

期权价格为什么会变化？

  • 标的价格（市场价）。

    标的价格上涨，则认购期权价格上涨，而认沽期权价格下跌。苹果股票从130涨到150，看涨期权赚的更多，所以期权价格也会涨。而同样若苹果股票从130涨到150，买了看跌期权的人大概率不能行权，为了少亏点权利金，就会降价卖看跌期权，自己少赔点，让哪些依然看跌的人接盘。

  • 行权价

    行权价高，则认购期权的价格越低，认沽期权的价格越高。同样的高行权价，看涨的人是以行权价买，所以大概率不会行权，除非市场价比行权价还高很多（不大可能，行权价本身已经是看涨的了）；但看跌的人是以行权价卖，这样非常容易赚钱。

    • 认购期权 = 看涨期权
    • 认沽期权 = 看跌期权
    • 行权价格 = 执行价格
  • 合约到期期限

    对于期权来说，期权到期期限越长，期权买方获利的可能性越大，期权卖方须承担的风险也越大，因此期权价格越高。从另外一个角度看，期权到期期限越长，期权的时间价值也就越高，这是要特别注意的期权价格影响因素。

  • 标的证券价格波动率（股票价格波动）。

    标的证券价格的波动率是用来衡量标的证券未来价格变动不确定性的指标，简单地说，标的证券价格波动率越高，期权合约到期时成为实值期权的可能性就越高，因此相应合约具有更高的价格。（因为期货价格一般与当前执行价格距离较大）

    • 实值期权：能挣钱，比如期货价格<执行价格的认购期权，或期货价格>执行价格的认沽期权。
    • 平值期权：期货价格 = 执行价格
    • 虚值期权：不会行权的，期货价格>执行价格的认购期权，或期货价格<执行价格的认沽期权。
  • 市场无风险利率。

    认购期权具有融资买入标的证券的性质，市场无风险利率越高，意味着融资成本越高，标的价格越容易涨，所以认购期权的价格就越高。

期权价格怎么定

期权价格是由买卖双方竞价产生的。期权价格分成两部分，即内涵价值和时间价值。期权价格=内涵价值+时间价值。

内涵价值

内涵价值是立即执行期权合约时可获取的利润。只有“实值期权”具有内涵价值，“平值期权”内涵价值为零，“虚值期权”无内涵价值。

比如：期货价格100元，执行价格90元的看涨期权，内含价值10元，如果此时权利金15元，则说明时间价值为5元。

再比如：期货价格90元，执行价格100元的看涨期权，是虚值期权，如果此时权利金为5元，则说明时间价值为5元。

一般来说，在其它条件一定的情况下，到期时间越长，期权的时间价值越大。在期权临近到期的时候，时间价值逐渐降低，在到期日当天，时间价值全部转变为内含价值。

期权是跟谁购买？做市商

注意：买call option的对手可不是买put option的人哦，他们俩都是买方，其共同的对手是做市商。

做市商：指在证券市场上，由具备一定实力和信誉的独立证券经营法人作为特许交易商，不断向公众投资者报出某些特定证券的买卖价格（即双向报价），并在该价位上接受公众投资者的买卖要求，以其自有资金和证券与投资者进行证券交易。

期货

套期保值

某一时间点，在现货市场和期货市场对同一种类的商品同时进行数量相等但方向相反的买卖活动，即在买进或卖出实货的同时，在期货市场上卖出或买进同等数量的期货，经过一段时间，当价格变动使现货买卖上出现盈亏时，可由期货交易上的亏盈得到抵消或弥补。从而在“现”与“期”之间、近期和远期之间建立一种对冲机制，以使价格风险降低到最低限度。

例子

猪肉市场价10000，期货价9000，投机者如果看多猪肉，则可以做多猪肉，买入10手猪肉期货。如果到期猪肉涨到了现货20000，期货18000，那他可以18000卖出10手猪肉期货，实现平仓，赚10*(18000-9000)=9w。

但对猪肉养殖场来说，他们手头有猪肉，如果他们也看涨猪肉，也可以做多猪肉，买10手猪肉期货。如果到期猪肉涨到了现货20000，期货18000，那他可以18000卖出10手猪肉期货，实现平仓，在期货市场是赚10*(18000-9000) = 9w，另外现货市场上也可以赚到10×(20000-10000)=10w。（但只有SB这么玩，如果猪肉跌了，他在现货市场和期货市场都要赔钱）

如果养殖场看空猪肉，则可以先10000卖出10手生猪期货，如果到时候真跌到了现货8000和期货8200，他现货市场会亏10×(10000-8000)=20000；但在期货市场，如果他以8200元买入10手生猪期货进行平仓，可以赚10×(9000-8200)=8000元。这样一对冲他就只亏12000元。

期货合约

3个月后以10000块钱买卖生猪，合约金5%.

做多和做空

做多者觉得某个商品未来会涨价，他买入期货。

做空者觉得某个商品未来会降价，他卖出期货。

  • 3个月后，生猪期货涨到了20000，为了平仓，做多的人卖出价值20000的期货，挣10000。而做空者要买入价值20000的期货，亏10000。
  • 3个月后，生猪期货跌到了5000，为了平仓，做多的人卖出价值5000的期货，亏5000。而做空者要买入价值5000的期货，挣5000。
